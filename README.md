Programação nas Escolas 🏫💻
Este repositório é o resultado de uma investigação aprofundada utilizando o NotebookLM, focada na integração do pensamento computacional e da lógica de programação no ambiente escolar.

📌 1. Contexto e Objetivos
Contexto
A alfabetização digital tornou-se uma competência essencial. No entanto, ensinar programação nas escolas vai além de digitar códigos; trata-se de ensinar os alunos a pensarem de forma estruturada. Escolhi este tema para entender como a educação brasileira está se adaptando a essa necessidade e quais as melhores práticas pedagógicas (como o Scratch e a computação desplugada).

Objetivos de Estudo
Explorar como a BNCC (Base Nacional Comum Curricular) aborda a computação.

Identificar metodologias para ensinar programação sem depender de laboratórios de informática caros.

Criar um guia de referência para educadores que desejam iniciar esse processo.

[Ensino de Programação com a Linguagem Python]: (https://www.youtube.com/watch?v=hAbstZQRqFk)) - Essa palestra tem o objetivo de criar uma discussão sobre o ensino de programação utilizando a linguagem Python. Como a linguagem pode ser utilizada nos cursos introdutórios de programação? Qual a experiência do ministrante com o uso da linguagem em cursos técnicos de nível médio (2º ano do ensino médio)? Quais estratégias podem ser utilizadas estimular o interesse dos alunos no aprendizado de programação?Além de apresentar conceitos básicos da linguagem, e vários exemplos para ilustrar o tema (ensino de programação), serão apresentadas ferramentas utilizadas para o estudo da linguagem Python. O público-alvo será professores de ensino fundamental e/ou médio de escolas públicas.Essa palestra justifica-se no fato de vários professores de escolas públicas terem interesse em aplicar novos conteúdos em aula, principalmente voltado à programação de computadores, e às vezes não terem o conhecimento de ferramentas simples que podem ser utilizadas no dia a dia. 

[A Importância do ensino da programação para as crianças - Palestra Mitchel Resnick]: (https://www.youtube.com/watch?v=XC13g1ZjaEI)) - Mitchel Resnick, professor do MIT palestra no Brasil sobre a importância do ensino de programação para jovens e adolescentes

📖 4. Miniguia de Estudo (Entrega Final)
Resumo Estruturado
O Pensamento Computacional: Dividido em quatro pilares: Decomposição, Reconhecimento de Padrões, Abstração e Algoritmos.

A Abordagem Brasileira: A integração obrigatória via BNCC e os desafios de infraestrutura.

Ferramentas Recomendadas: Scratch (Visual), Blockly, e atividades desplugadas (Papel e Caneta).

Glossário de Conceitos
Algoritmo: Sequência finita de instruções para resolver um problema.

Computação Desplugada: Ensino de conceitos de computação sem o uso de dispositivos eletrônicos.

Sintaxe: O conjunto de regras que define como o código deve ser escrito.
